搭建博客之旅

Dec 1, 2019 | 搭建博客

记录一下我搭建博客的过程，方便以后再次使用相关的命令（主要给自己看）

使用的软件：

  1. node
  2. hexo
    部署的地方：
  3. gihub
  4. gitee
  5. coding（腾讯家的）

    下载node，hexo

    下载node，百度搜索就是，官网如下：


    安装好之后，找个合适的位置，在cmd里下载安装hexo

创建博客

同样cmd里操作 hexo n "博客名"

则在你安装hexo的source/_posts目录下，出现一个md文件，和一个文件夹




一个是写博客的，一个是放博客中图片的文件夹

至于在博客中怎么放图片的格式，网上markdown格式搜一下就行

将博客托管

如果你有服务器还好，没有的话。可以将博客托管在github，gitee，coding上。个人选择coding，毕竟github在国内访问有点慢。

设置托管的路径
在hexo安装目录下的_config.yml文件
找到这个位置

照下图填写（我这里填了三个，也就是说我上传到了三个地方，但我域名绑定的是coding上的）


上传博客的话
同样在cmd中进入hexo安装的目录下

注意

如果博客中用到了图片，将图片上传到coding上会显示不出来。

有可能的原因是你的图片路径使用了中文，而上传过程中，会将中文编码，而造成问题
如你使用的是这样的路径


但在托管中的是这样的


所以你的博客网站上无法正常显示图片。

可以在hexo文件目录的任何地方重新创建一个英文或以日期为标准的文件夹，装门存放一些博客的图片文件

这样，在托管中的图片文件的路径就能正常显示了
